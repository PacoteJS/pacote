Shareable ESLint configuration for Jest.

## Installation

```bash
yarn add --dev @pacote/eslint-config-jest eslint
```

## Usage

Include the configuration in your project's `.eslintrc` file:

```json
{
  "extends": ["@pacote/eslint-config-jest"]
}
```
