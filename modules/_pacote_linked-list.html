    @pacote/linked-list

    Immutable linked lists.

    Installation

    yarn add @pacote/linked-list

    Example

    import { listOf, sort } from '@pacote/linked-list'

    sort(listOf(3, 2, 1)) // => [1, [2, [3, undefined]]]
