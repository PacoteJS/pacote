Class LRUCache<K, V>

Implements a Least-Recently Used cache based on a Map.

Example

import { LRUCache } from '@pacote/lru-cache'

const cache = new LRUCache(1000)

cache.set('key', 'value')
cache.get('key') // => 'value'
